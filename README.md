# QualityDisser - เว็บไซต์บริการสนับสนุนทางวิชาการ

นี่คือเว็บไซต์บริการสนับสนุนทางวิชาการมืออาชีพที่สร้างขึ้นเป็นหน้า HTML แบบคงที่พร้อม Tailwind CSS และ JavaScript แบบกำหนดเอง

## คุณสมบัติ

- การออกแบบที่ตอบสนองได้บนอุปกรณ์ทุกชนิด
- UI ที่ทันสมัยพร้อมภาพเคลื่อนไหวและการเปลี่ยนผ่านอย่างราบรื่น
- ส่วนคำถามที่พบบ่อยแบบโต้ตอบด้วยฟังก์ชัน accordian
- รองรับโหมดมืดพร้อมการบันทึกค่าตั้งอัตโนมัติ
- การนำทางแบบเลื่อนอย่างราบรื่น
- องค์ประกอบภาพเคลื่อนไหวที่ปรากฏเมื่อเลื่อนดู
- การแสดงบริการทางวิชาการอย่างมืออาชีพ

## โครงสร้างไฟล์

```
new_static_website/
├── index.html          # ไฟล์ HTML หลัก
├── css/
│   └── style.css       # สไตล์ CSS แบบกำหนดเอง
├── js/
│   └── script.js       # ฟังก์ชัน JavaScript แบบกำหนดเอง
├── images/             # ไดเรกทอรีรูปภาพ (สำหรับใช้ในอนาคต)
└── README.md           # ไฟล์นี้
```

## เทคโนโลยีที่ใช้

- HTML5
- Tailwind CSS (ผ่าน CDN)
- CSS แบบกำหนดเอง
- JavaScript แบบ Vanilla
- Google Fonts
- Material Symbols

## วิธีใช้งาน

เพียงเปิด `index.html` ในเบราว์เซอร์ที่ทันสมัยใด ๆ เพื่อดูเว็บไซต์ ไม่จำเป็นต้องใช้เซิร์ฟเวอร์

## การปรับแต่ง

- แก้ไข `css/style.css` เพื่อปรับสี ระยะห่าง และองค์ประกอบภาพอื่น ๆ
- อัปเดต `js/script.js` เพื่อเปลี่ยนพฤติกรรมโต้ตอบ
- แก้ไข `index.html` เพื่อเปลี่ยนเนื้อหา บริการ คำรับรอง เป็นต้น

## การรองรับเบราว์เซอร์

เว็บไซต์นี้เข้ากันได้กับเบราว์เซอร์ที่ทันสมัยทั้งหมด (Chrome, Firefox, Safari, Edge)
